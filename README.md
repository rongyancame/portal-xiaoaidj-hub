# portal-xiaoaidj-hub

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为页面资源的管理与分发点。

## 项目简介

本仓库主要存放各类独立 HTML 页面，这些页面可能是工具、演示、信息展示或其他用途。所有页面均为静态内容，不依赖后端服务，可以直接在浏览器中打开使用。

## 目录说明

```
portal-xiaoaidj-hub/
├── pages/          # 存放 HTML 页面文件
├── assets/         # 存放页面所需的资源文件（图片、样式、脚本等）
└── README.md       # 本说明文件
```

- `pages/`：每个子目录或文件对应一个独立页面。
- `assets/`：共享资源，按需引用。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，命名清晰，便于识别。
- 页面间无相互依赖，可单独部署或使用。
- 本仓库不跟踪页面内容的更新历史，但可通过 Git 提交记录查看变更。

## 维护说明

- 欢迎提交 issue 报告问题或提出改进建议。
- 如需添加新页面，请将 HTML 文件放入 `pages/` 目录，并在必要时更新 `assets/`。
- 请勿在页面中嵌入恶意代码或违反开源社区规范的内容。
- 本仓库遵循通用开源协议，具体请参见仓库根目录的 LICENSE 文件。

## 使用方式

1. 克隆本仓库到本地。
2. 使用浏览器直接打开 `pages/` 下的 HTML 文件。
3. 或通过任意静态文件服务器托管整个仓库。

## 其他

本仓库仅用于技术归档与共享，不保证页面内容的长期可用性。使用者应自行判断页面内容的适用性。
